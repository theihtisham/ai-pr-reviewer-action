# Ai Pr Reviewer

AI-powered GitHub Action for automated PR reviews

## Installation

```bash
npm install @theihtisham/ai-pr-reviewer
```

## Usage

```typescript
import { } from '@theihtisham/ai-pr-reviewer';
```

## Features

- Written in TypeScript with full type definitions
- Zero external dependencies (minimal footprint)
- Event-driven architecture
- Comprehensive test suite
